# Monitor de Usuários do GitHub

Projeto desenvolvido em Python utilizando a API do GitHub e SQLite para consultar e armazenar informações de usuários.

## Funcionalidades

- Consultar usuários pela API do GitHub
- Salvar usuários no banco de dados
- Evitar usuários duplicados
- Listar usuários salvos
- Remover usuários

## Tecnologias utilizadas

- Python
- Requests
- SQLite
- API do GitHub
- Git
- GitHub

## Objetivo

Projeto criado para praticar:
- Consumo de APIs
- Manipulação de JSON
- Banco de dados SQLite
- CRUD
- Organização de código com funções
