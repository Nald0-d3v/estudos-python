# Sistema de Candidaturas

Projeto desenvolvido em Python utilizando SQLite para gerenciar candidaturas de emprego.

## Funcionalidades

- Adicionar candidaturas
- Listar candidaturas
- Atualizar status
- Remover candidaturas
- Validação de salário
- Validação de datas

## Tecnologias utilizadas

- Python
- SQLite
- Git
- GitHub

## Objetivo

Projeto criado para praticar:
- CRUD
- SQL
- Funções
- Tratamento de erros
- Integração entre Python e banco de dados
